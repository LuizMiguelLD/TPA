<h1>Técnicas de Programação e Algoritmos (TPA)</h1>
<p>Técnicas de Programação e Algoritmos ou TPA, é uma matéria que estuda métodos para criar algoritmos para o código. Ou seja, através da lógica, criar uma série de passos lógicos (algoritmos) que seram transformadas em código.</p>
<p>Então aqui nesse repositório possui os códigos que fiz através de algoritmos. Provenientes de séries de exercícios que foram feitos ao decorrer do ano.</p>
